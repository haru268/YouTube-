# 動画管理システム

YouTube動画を管理するためのWebアプリケーションです。

## 機能

- ログイン認証
- チャンネル情報の管理
- 動画予定の管理（タイトル、No、種類、冒頭内容、ナレーション内容、投稿状況）
- 投稿済み動画の確認（手動入力）
- パスワード変更
- エクセル風テーブル表示（インライン編集、ソート機能）
- 検索・フィルタリング機能
- 一括操作機能
- テンプレート機能
- 下書き保存機能
- 統計ダッシュボード
- カレンダー表示

## 必要な環境

- Node.js (v14以上)
- npm

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env`ファイルを作成し、以下を設定してください：

```
PORT=3000
SESSION_SECRET=your-secret-key-change-this
YOUTUBE_API_KEY=your-youtube-api-key-here
INITIAL_USERNAME=admin
INITIAL_PASSWORD=admin123
```

### 3. YouTube APIキーの取得方法

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 新しいプロジェクトを作成（または既存のプロジェクトを選択）
3. 「APIとサービス」→「ライブラリ」から「YouTube Data API v3」を有効化
4. 「APIとサービス」→「認証情報」から「認証情報を作成」→「APIキー」を選択
5. 作成されたAPIキーを`.env`ファイルの`YOUTUBE_API_KEY`に設定

### 4. データベースの初期化

```bash
npm run init-db
```

### 5. サーバーの起動

```bash
npm start
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## 初期ログイン情報

- ユーザー名: `admin` (`.env`ファイルで変更可能)
- パスワード: `admin123` (`.env`ファイルで変更可能)

**重要**: 初回ログイン後、必ずパスワードを変更してください。

## 使い方

1. ログイン後、まず「設定」画面でチャンネル情報を設定します
2. 「動画予定」画面で、これから作成する動画の予定を管理できます
3. 「投稿済み動画」画面で、投稿済みの動画情報を手動で追加・管理できます
4. 「統計」画面で、動画の統計情報を確認できます
5. 「カレンダー」画面で、動画のスケジュールを視覚的に確認できます

## プロジェクト構造

```
動画管理/
├── public/              # フロントエンドファイル
│   ├── css/            # CSSファイル
│   ├── js/             # JavaScriptファイル
│   └── *.html          # HTMLファイル
├── scripts/            # スクリプト
│   └── init-db.js      # データベース初期化
├── server.js           # サーバーファイル
├── videos.db           # SQLiteデータベース（自動生成）
├── package.json        # 依存パッケージ
└── README.md           # このファイル
```

## 注意事項

- データベースファイル（`videos.db`）はプロジェクトフォルダ内に作成されます
- `.env`ファイルには機密情報が含まれるため、Gitにコミットしないでください
- YouTube API機能は現在無効化されています（課金が必要なため）
- 視聴回数やいいね数は手動で入力してください

## トラブルシューティング

### データベースが見つからないエラー

`npm run init-db`を実行してデータベースを初期化してください。

### YouTube APIエラー

- APIキーが正しく設定されているか確認してください
- YouTube Data API v3が有効化されているか確認してください
- APIキーの使用制限を確認してください
